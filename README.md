## Installation

[![NPM](https://nodei.co/npm/every-react-layout.png?downloadRank=true&stars=true)](https://nodei.co/npm/every-react-layout/)

Via NPM or Yarn:
```sh
npm install every-react-layout
yarn add every-react-layout
```

npm run dev
